# ttm-bmi-hypertension
R pipeline for a prospective cohort study of baseline BMI category and incident treated hypertension in Australian men (Ten to Men linked with PBS/MBS; n=12,742, 11-year follow-up). Cox models, multiple imputation, bootstrap C-index, decision curve analysis, E-values, competing-risks sensitivity. Manuscript under revision (Chron Dis Transl Med).
