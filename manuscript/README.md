# Manuscript

The manuscript and response-to-reviewers documents are maintained separately
from this code repository. This file maps each manuscript table and figure to the
script that generates it, for reviewers and for reproducibility.

## Figure → script mapping

| Figure | Description | Generated by |
|---|---|---|
| Figure 1 | Kaplan–Meier hypertension-free survival by BMI category | `R/06_figures.R` |
| Figure 2 | Forest plot of BMI hazard ratios (M1–M3) | `R/06_figures.R` |
| Figure 3 | C-index of nested models with bootstrap CIs | `R/06_figures.R` |
| Figure 4 | Decision curve analysis with bootstrap confidence ribbons | `R/11_dca_bootstrap.R` |
| Figure 5 | Decile-based calibration plot | `R/13_calibration_deciles.R` |
| Figure 6 | Schoenfeld residual plots for BMI contrasts | `R/10_schoenfeld_plot.R` |

> Note: `R/06_figures.R` also produces earlier point-estimate versions of the DCA
> and calibration figures (`fig4_dca.png`, `fig5_calibration.png`). These were
> **superseded** in the final manuscript by the bootstrap-CI DCA (Figure 4,
> script 11) and the decile calibration (Figure 5, script 13).
